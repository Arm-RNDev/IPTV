# Installation

1. Clone the repository
 ### git clone https://github.com/Arm-RNDev/IPTV
 ### cd IPTV

2. Install dependencies
 ### npm install
  or
### yarn install
3. iOS setup (macOS only)
  ### cd ios && pod install && cd ..
4. Start Metro
  ### npm start

5.Running the App

   Android
  ###  npm run android

   IOS
   ### npm run ios

#Requirements

  Node 20+
  Android Studio or Xcode
  React Native CLI environment
